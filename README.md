# Mi-red-social
Testeo de codigo y diseño UI/UX y coneccion de base de datos en entorno LAMP.

Concepto de una red social cuyo fin central es publicar audios entre amigos.

En el siguiente enlace se puede descargar la base de datos con sus tablas que se uso para este proyecto:

https://www.mediafire.com/file/mo5p4yyfm137eyw/base_de_datos.rar/file

Se uso My SQL workbench (para generar las tablas), phpmyadmin (para visualizar la base de datos y como un complemento mas comodo para asignar las llaves foraneas y demas actividades) , Xampp (como servidor local), y Visual Studio Code (para el codigo).
